# GOOGLE_HPS_SmartHome
2022 Taiwan Google Hardware Sprint Team 9

## Sensor Node
Through the HC-08 bluetooth module, Arduino sends sensor data like temperature, relative moisture, and light intensity to Raspberry Pi. Then, Raspberry Pi saves sensor data as txt file, and control the wireless Tapo P110 switches to open the home appliances according to the data collected.
